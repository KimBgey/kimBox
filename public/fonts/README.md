# Fonts

Placer ici les fichiers de polices personnalisées :

- `Sunroll.woff2` — police display (grands titres)
- `Hagrid-Regular.woff2` — texte courant 400
- `Hagrid-Medium.woff2` — texte courant 500
- `Hagrid-Bold.woff2` — texte courant 700

Une fois les fichiers présents, décommenter les imports dans `app/layout.tsx`.
